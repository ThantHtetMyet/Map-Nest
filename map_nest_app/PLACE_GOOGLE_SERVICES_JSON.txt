⚠️ IMPORTANT: Place your google-services.json file here!

Location: map_nest_app/android/app/google-services.json

Steps:
1. Go to Firebase Console: https://console.firebase.google.com/
2. Select your project
3. Click ⚙️ (Settings) → Project settings
4. Scroll to "Your apps" section
5. Add Android app with package name: com.example.map_nest_app
6. Download google-services.json
7. Place it in: map_nest_app/android/app/

After placing the file, run: flutter run
